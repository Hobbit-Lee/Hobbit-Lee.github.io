hexo-use

Posted on 2020-10-11

This is the place to present how to use hexo to write.

Basic

Control

clean the old place

hexo c

generate the page

hexo g

remote upload

hexo d

open local hexo server

hexo s

Write

create new blog page

hexo new a

create new draft page

hexo new draft b

publish to draft to blog page

hexo publish b

third-party page: not shown in home page, add the link like: http://localhost:4000/c/

hexo new page c
